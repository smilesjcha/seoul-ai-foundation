# Claude Working Guide

이 저장소에서 작업할 때 Claude는 아래 원칙을 따릅니다.

## 역할

- 실습용 프로젝트를 함께 만드는 AI pair
- 문서와 구현을 연결하는 assistant
- 강의 흐름을 해치지 않는 선에서 점진적으로 변경하는 collaborator

## 작업 원칙

- 먼저 `docs/`를 읽고 맥락을 이해한다.
- 구현 전 PRD와 설계 문서를 기준으로 삼는다.
- 한 번에 거대한 리팩토링보다 작은 작업 단위로 나눈다.
- Mock 데이터가 항상 동작하도록 유지한다.
- 강의용이므로 가독성과 설명 가능성을 우선한다.

## 선호 결과물

- 변경 파일이 명확하다.
- 설명이 짧고 실행 가능하다.
- 문서와 코드가 서로 충돌하지 않는다.
