# 강의 개요

## 강의명

개발자에서 AI Native로: Claude Desktop으로 완성하는 업무 생산성 혁신

## 강의 대상

- AI 도구를 실제 업무에 붙이고 싶은 PM, 개발자, 기획자
- Claude Desktop을 문서 작성 도구를 넘어 "실행 파트너"로 쓰고 싶은 실무자
- 실습형으로 AI Native 제품 개발 흐름을 경험하고 싶은 교육 참여자

## 핵심 메시지

이 강의의 핵심은 도구 소개가 아닙니다. Claude Desktop을 통해 문제 정의, PRD 작성, 설계, 구현, 검토를 하나의 흐름으로 연결하는 "업무 운영 방식"을 체득하는 데 있습니다.
